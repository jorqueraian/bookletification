# bookletification
Makes booklets out of pdfs

Running the code `python bookification.py` will give you input options

```
It seems like you need help! <> indicates a required field, and [] indicates an optional field.
bookification <input pdf file> [page scaling = 1.1] [blanks at start = 0] [blanks at end = 0]
```
When an input pdf is selected you will be asked how many sheets of paper should be used in each signature. If you wish to only have one signature, leave blank.


To run the code do the following
```python bookification.py file.pdf```

When printing, print double sided with the option that flips pages on the short edge!
